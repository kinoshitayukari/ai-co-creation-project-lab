ポートフォリオ実例に使うもの

実例1: BTCUSD時間帯別大変動分析

BTCUSDの価格データを使い、日本時間で大きな変動が出やすい時間帯を集計。

見せ方:

  • 入力: BTCUSD M1データ
  • 処理: H1変換、3時間後・6時間後の大変動率を集計
  • 出力: 時間帯別テーブル、Markdownレポート

実例2: BTCUSD no-lookaheadバックテスト集計

BTCUSDのバックテスト結果を、未来データ参照を避ける設計で検証。

見せ方:

  • 入力: BTCUSD価格データ
  • 処理: シグナル生成、コスト反映、トレード集計
  • 出力: PF、勝率、最大ドローダウン、年別集計

実例3: Hyperliquid fills集計

Hyperliquidの約定履歴をBTCに絞り、close / open、buy / sell、closedPnl、推定エントリー時間を整理。

見せ方:

  • 入力: fills CSV
  • 処理: BTC抽出、損益集計、推定エントリー価格計算
  • 出力: 集計CSV、再現メモ、分析レポート
